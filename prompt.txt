Create a web-based terminal interface called "kimitui" that lets users chat with AI models on Cloudflare Workers AI through a browser.
The UI looks and feels like a terminal — dark background, green monospace text, a scrolling chat area, and an input prompt at the bottom.

Technical stack:
- Frontend: Next.js 16 (App Router) with Tailwind CSS
- Package manager: yarn
- Backend: Next.js API routes call Cloudflare Workers AI API (OpenAI-compatible) directly via fetch
- Env vars: CLOUDFLARE_ACCOUNT_ID and CLOUDFLARE_API_TOKEN in .env

Features:
- Terminal-style chat interface with streaming responses (SSE from Cloudflare API)
- Slash commands: /help, /models (list + select by number), /model <name>, /info <model>, /clear, /stop
- Esc key to abort streaming
- Model selection persists via localStorage
- Header bar showing current model name and thinking indicator
- Model list shows numbered entries, bare number input after /models selects that model

API actions (handled by src/app/api/chat/route.ts): list_families, list_models, model_info, chat, stream_chat — all via NDJSON responses.
